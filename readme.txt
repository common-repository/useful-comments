== Description ==

Set useful comments for post each.

== Installation ==

Set useful comments for post each. 
Upload `useful_comments` to the `/wp-content/plugins/` directory.
To show useful comments put <?php wp_uc_flash(); ?> in your templates~ 
Navigate to `Editing Comment` page to configure or cancel useful comments.
